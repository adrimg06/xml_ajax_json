Ejercicio 7 - Preguntas sobre XML

¿Cómo se ven los comentarios en XML? (Sección 2.5)

Los comentarios en XML están delimitados por <!-- y -->. No pueden contener la secuencia -- dentro del comentario.

Ejemplo: <!-- Este es un comentario en XML -->

¿Puede repetirse un atributo en el mismo elemento? (Sección 3.1)

No, un atributo no puede repetirse en el mismo elemento. Cada atributo debe tener un nombre único dentro de un elemento.

¿Dónde se establece que los nombres de los elementos en las etiquetas de apertura y cierre deben coincidir?

Esto se establece en la Sección 3.1 del estándar XML 1.0. Especifica que los nombres de las etiquetas de apertura y cierre deben coincidir exactamente para que el documento sea válido.
